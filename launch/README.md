# Lumina — Founding-100 Beta Launch Kit

Ready-to-run assets for the v2.0 English-First Beta (Sprint 5 of `ROADMAP.md`).
US-first / English-first GTM · 100 invited creators · 14-day validation window.

## Files

| File | Purpose |
|---|---|
| `beta-outreach-templates.md` | Cold outreach scripts (IG/TikTok DM, Reddit DM, Reddit comment, Discord DM, Email, X DM) + per-channel send caps and hard rules. |
| `beta-offer-copy.md` | Founding-100 offer copy in three lengths (DM / email / landing-page hero) plus the locked offer terms. **Send only after a creator replies "yes".** |
| `beta-validation-tracker.csv` | Header row for the 14-day validation tracking sheet — one column per v2.0 success metric. Import into Sheets / Airtable / Linear. |

## v2.0 success metrics this kit measures

- **Activation:** invite → activation latency, Style Twin training completion, similarity ≥ 0.998 gate.
- **Engagement:** swarm runs per day (1 / 7 / 14), publish-completion rate, one-tap publish usage.
- **Monetization:** brand-deal acceptance, affiliate auto-match rate, gross earnings at d7 / d14, monetization rate %.
- **Payouts:** payout rail chosen (Stripe Connect / PayPal instant), first payout latency, amount.
- **Compliance:** shield blocks per creator (CCPA · EU AI Act · COPPA · FTC · GDPR).
- **Virality:** referral codes shared, referrals signed up, bounty paid.
- **Voice of customer:** day-7 + day-14 NPS and one qualitative quote each.

## Kill criteria (Sprint 5, v2.0 ROADMAP)

After 200 cumulative signups, pivot if:
- Swarm-run completion rate **< 25 %**, **or**
- Monetization rate (creators with ≥ $1 attributed earnings) **< 15 %**.

`kill_criterion_breached` and `founder_intervention_at_utc` columns in the tracker exist to catch this in real time.
