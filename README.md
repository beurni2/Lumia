## The Nuclear Moat

- **On-device quantized multimodal swarm** — Llama 3.2 11B Vision + Mistral 7B + Qwen 3.5 9B hybrids (4-bit/8-bit). Runs on any phone with 8 GB+ RAM (iOS 17+, Android 14+).
- **<90s end-to-end inference** — script → storyboard → edit → export, fully on-device. Selective edge-cloud burst (<5s additional) for complex renders.
- **Personal Style Twin** — one-time 10-video upload produces a permanent 99.8% voice/aesthetic clone. Retrains instantly on every new video. Encrypted on-device.
- **Cultural Voice Pack system** — modular on-device packs the Twin layers in instantly. Day-1 packs: **US Gen-Z humor, US lighting/aesthetic, US slang**, plus UK / CA / AU / IN / PH / NG. SEA/LATAM packs land in Phase 1.
- **4-agent collaborative swarm** — Ideator (English trends + US/EN platform quirks), Director (storyboards + hook optimization), Editor (visuals, captions, effects, music), Monetizer (deal matching + pitch generation), all coordinated by an Orchestrator via an internal memory graph.
- **Closed-loop monetization** — Lumina detects affiliate opportunities, generates brand pitch decks, drafts brand outreach, tracks and deposits earnings via **Stripe Connect + PayPal instant payouts** (US bank accounts first). **10% performance fee only on the incremental revenue Lumina creates.**
- **Compliance Shield + multi-platform redundancy** — TikTok, Instagram Reels, YouTube Shorts as Day-1 priority with full API integration; Kwai/GoPlay-ID/Kumu-PH policy packs ship pre-wired so Phase 1 SEA/LATAM layering is a config flip, not a refactor.
- **Compliance & privacy from day one** — full **CCPA, EU AI Act, COPPA, FTC disclosure, GDPR** readiness in MVP. The on-device moat is the #1 global differentiator.

## Contribution Guidelines

Lumina is built to a **non-negotiable standard**. Every contribution must clear the same bar.

### The Constitution

1. **Vision-aligned to v2.0 or rejected.** Every PR must declare which spec capability it advances. Drive-by features that do not serve the immutable v2.0 blueprint are closed without discussion.
2. **Privacy-first, always.** Zero raw user footage leaves the device without explicit per-action consent. No analytics that fingerprint creators. No third-party SDKs that exfiltrate. CCPA + EU AI Act + COPPA + FTC + GDPR readiness from MVP.
3. **On-device first, cloud only when justified.** Any cloud call must document why on-device cannot serve the use case within budget.
4. **English-first cultural authenticity is P0.** Copy, mock data, and trend logic for US/UK/CA/AU/IN/PH/NG markets ship as default. Phase 1 SEA/LATAM packs are reviewed by a regional contributor before merge.
5. **Tests are not optional.** Every public endpoint, hook, component, and utility ships with tests. Major features include integration + E2E coverage.
6. **No placeholders, no shortcuts, no compromise.** Senior-engineer-grade code or it does not merge.
