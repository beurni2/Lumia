PART 2 — TEMPLATE RULES

1. Each HookLanguageStyle MUST use its own templates
2. Generic templates are fallback only
3. Penalize templates that work across too many scenarios
4. Prefer:
   - short
   - specific
   - emotionally sharp
   - slightly awkward

---

PART 3 — SCROLL-STOPPING LAYER (CRITICAL)

Add a new scoring boost:

scrollStopScore (0–10)

Increase score if hook:
- creates tension
- feels abrupt
- is incomplete / fragment
- implies something went wrong
- is emotionally sharp
- breaks expected sentence structure

---

BOOST CONDITIONS:

+3 if fragment or abrupt
+2 if emotionally charged
+2 if unusual structure
+2 if not a full sentence
+1 if highly specific

---

PENALTIES:

-3 if generic sentence
-3 if reusable across scenarios
-2 if “safe” phrasing
-2 if over-explained

---

PART 4 — HARD REJECTION

Reject hook if:
- starts with overused patterns:
  - "the way I"
  - "why did I"
  - "I really thought"
- structure identical to recent hooks
- easily swappable across 3+ scenarios

---

PART 5 — GOAL

Hooks should feel like:
- interruptions
- thoughts mid-action
- emotionally real
- slightly messy
- human

NOT:
- clean
- generic
- explanatory
- templated

---

SUCCESS CRITERIA

When reading 3 hooks:
They should feel like:
"these are completely different thoughts"

Not:
"same sentence, different object"
