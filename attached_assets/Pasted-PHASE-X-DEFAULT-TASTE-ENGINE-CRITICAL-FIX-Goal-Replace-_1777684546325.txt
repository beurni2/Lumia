Replace generic idea generation with a strong default comedic taste derived from curated hooks.

Create DefaultTasteProfile based on:

- self_betrayal (highest weight)

- confident_vs_collapse

- confession

- lowercase
- conversational
- deadpan
- short

Targets:
- high specificity
- medium absurdity
- high intensity
- very low softness

---

PART 2 — PREMISE-FIRST GENERATION (MANDATORY)

Refactor generation pipeline:

OLD (REMOVE):
hook → scenario → filming (independent)

NEW (REQUIRED):
ComedyDNA → Premise → derive everything

Steps:

1. sample ComedyDNA using DefaultTasteProfile weights
2. generate ONE strong comedic premise
3. derive:
   - hook
   - whatToShow
   - howToFilm

STRICT RULE:
All outputs must come from SAME premise.

If mismatch detected → reject.

---

PART 3 — COMEDY VALIDATION (HARD GATE)

Reject any idea if:

- no clear contradiction
- no tension (intention vs reality)
- reads as observation instead of joke
- safe / generic tone
- similar to last 5 ideas

Only allow:
strong, specific, tension-based comedy

---

PART 4 — ANTI-LOOP SYSTEM

Track:
- last 5 idea domains
- last 5 mechanisms

Penalize:
- same domain repetition
- same structure repetition

Force variation across:
- life domains (money, health, work, social, habits)
- comedic mechanisms

---

PART 5 — ALIGNMENT GUARANTEE

Before returning output:

Verify:

- hook matches premise
- scenario expresses premise
- filming executes scenario

If any mismatch:
→ regenerate
